### How to use this for your scraping plan

1. **Create a text file** (e.g., `sietk_urls.txt`) and paste all these URLs.  
2. For each URL:
   - If it’s a **page** (`.php`): use your Firecrawl `scrape` (when you have credits) or a local HTML saver + parser.  
   - If it’s a **PDF**: download it and use a Python PDF text extractor (like `pypdf`) to convert to text.

3. For the chatbot, store for each document:
   - `markdown` or extracted text.  
   - `source_url` (the page or PDF URL).  
   - Category (About, Dept, Syllabus, etc.) if you want.

Because the site also has many small utility pages and possibly some hidden/internal ones, there may be more URLs than this list, but if you cover **home + about + all departments + exam cell + syllabus + placements + library + key PDFs**, your chatbot will already feel very complete for students and applicants.
